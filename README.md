# Simple Microblog
Blog based on [Clean Blog](https://github.com/BlackrockDigital/startbootstrap-clean-blog) template

![](http://i.imgur.com/YKPc9lb.png)

# Features
* Support for tags
* Interactive guest book
* REST API
* Support for RSS
* WYSIWYG markdown editor
* Clean, simple and responsive theme
