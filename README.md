# Simple Microblog
Blog based on Clean Blog template ([Preview](http://imgur.com/kRiysCm))

![](http://i.imgur.com/Tg4tTE3.jpg)

# Features
* Support for tags
* Interactive guest book
* Internationalization
* REST API
* Support for RSS
* WYSIWYG markdown editor
* Clean, simple and responsive theme
